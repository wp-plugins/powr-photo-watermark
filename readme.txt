== Installation ==
To install POWr Photo Watermark
<br><br>
1. Upload 'powr-photo-watermark.php' to the '/wp-content/plugins/' directory
<br>
2. Activate the plugin through the 'Plugins' menu in WordPress
<br><br>
To add POWr Photo Watermark as a widget, go to **Appearance->Widgets**, and drag POWr Photo Watermark to where you'd like it to appear.
<br><br>
To add Photo Watermark within a post or a page, simply write the shortcode **[powr-photo-watermark label='MY LABEL']** (including the brackets) where you'd like your Photo Watermark to appear. Then visit your page!
<br><br>
Want a more detailed tutorial? Visit [How to add Photo Watermark to your Wordpress site](http://www.powr.io/tutorials/how-to-add-photo-watermark-plugin-to-your-wordpress-site)
